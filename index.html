Christendom Budgeting

GitHub Database Setup

This web app opens and saves the real budget.sqlite3 file in GitHub. Use a private repo and a fine-grained token with Contents read/write.

Want to try it first? Demo mode keeps a sample budget only in this browser and does not use GitHub sync.

New User GitHub Sync Setup
  1. Create or sign in to a GitHub account.
  2. Create a new private repository, for example christendom-budget-sync.
  3. Create a fine-grained personal access token for only that repository.
  4. Give the token Contents: Read and write permission.
  5. Paste the token here and tap Load Repositories.
  6. Select the repository from the dropdown.
  7. Select the branch, usually main.
  8. Tap Open Database. Christendom Budgeting will save budget.sqlite3 in the root of that branch.

Keep the repository private and keep the token limited to this one budget repository.
